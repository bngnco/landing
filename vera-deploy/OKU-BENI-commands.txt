Bu paket, /vera sayfasını ve son CMS kapak görsellerinin
optimizasyonunu içeren commit'leri taşır. Codespaces'te yazma
yetkisi hazır geldiği için tek yapman gereken uygulayıp push'lamak.

Codespaces'i repo kök dizininde aç, bu zip'i oraya yükle ve unzip et.
Sonra AŞAĞIDAKİ YOLLARDAN BİRİNİ uygula.

───────────────────────────────────────────────────────────────
YOL A — BUNDLE (önerilen, en sağlam)
───────────────────────────────────────────────────────────────
  git checkout main
  git pull origin main
  git fetch ./vera.bundle claude/landing-page-animation-mobile-sipxgu
  git merge --no-edit FETCH_HEAD
  git push origin main

  # Not: "Already up to date" veya çakışma OLMAMALI. Çakışırsa
  # (araya yeni CMS commit'i girdiyse) YOL B'yi dene.

───────────────────────────────────────────────────────────────
YOL B — PATCH (yol A çakışırsa)
───────────────────────────────────────────────────────────────
  git checkout main
  git pull origin main
  git apply --3way --whitespace=nowarn vera-changes.patch
  git add -A
  git commit -m "Add Vera research page (/vera) + optimize CMS cover images"
  git push origin main
