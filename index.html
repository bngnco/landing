03 · HOW IT WORKS

Five signals. One verdict.

Content credentials, metadata forensics, neural face analysis, frequency spectrum and audio authenticity — fused into a single dynamic trust score.

01

Originality stamp

C2PA content credentials — the camera's signed receipt that proves a file came from a real device.

02

File fingerprint

EXIF, encoding consistency, frame-rate anomalies — the metadata trail every editor leaves behind.

03

Face review

Apple Vision face analysis frame-by-frame, looking for the subtle artifacts AI generators can't hide.

04

Motion check

Optical-flow and temporal consistency — objects must move like real objects do.

05

Pattern check

DCT frequency spectrum looking for the statistical fingerprint of diffusion and GAN models.

Advanced

Advanced deepfake detection
